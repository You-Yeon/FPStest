FPStest
=============
> FPS의 기본 기능들을 구현한 게임입니다.

📝 Introduction
------------
개인프로젝트 TPS 물총 게임인 'Aquarium'을 개발하기 전에,   
총 게임 개발을 익히기위해 FPS 게임 예제들을 참고하며 만든 학습 및 테스트용 프로젝트입니다.

:computer: Developerment skill
------------
- 언어 : C#
- 게임 엔진 : Unity v2019.1.8f1

:gun: Description
-----------
구현해본 기능은 총쏘기, 정조준, 반동, 산탄, 탄흔이 있습니다.
총쏘는 방식은 히트스캔방식입니다.
